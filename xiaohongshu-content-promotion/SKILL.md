# 小红书内容推广技能

## 概述

本技能用于在小红书平台上进行产品推广。通过搜索与产品相关的热门帖子，在帖子下方发布有针对性的评论，引导有需求的用户关注和使用推广的产品或服务。

## 使用场景

- 推广微信小程序、APP、产品服务等
- 在目标用户聚集的内容下方进行软性推广
- 需要在评论区进行精准营销的场景

## 前置要求

1. 确保 `xiaohongshu-mcp` 服务已配置并登录
2. 准备要推广的产品信息（名称、功能特点、目标用户群体等）
3. 准备已评论过的帖子记录文件路径（避免重复评论）

## 操作流程

### 步骤 1：搜索相关热门帖子

使用 `search_feeds` 工具搜索与产品相关的热门内容：

```json
{
  "keyword": "搜索关键词",
  "filters": {
    "sort_by": "最多收藏",  // 或"最新"、"最多点赞"等
    "publish_time": "半年内",
    "note_type": "不限"
  }
}
```

**技巧：**
- 使用产品目标用户会搜索的关键词
- 优先选择收藏量高、互动多的热门帖子
- 设置合理的筛选条件获取高质量内容

### 步骤 2：读取已评论记录

读取已评论过的帖子记录文件（通常为 JSON 格式），提取已评论的 `feed_id` 列表，用于后续过滤。

```json
{
  "commented_posts": [
    {"feed_id": "xxx", "title": "..."},
    ...
  ]
}
```

### 步骤 3：筛选未评论的新帖子

从搜索结果中过滤掉已评论的帖子，对剩余帖子逐一处理。

### 步骤 4：获取帖子详情（可选）

对于筛选出的新帖子，可以使用 `get_feed_detail` 获取详细内容，以便撰写更有针对性的评论。

```json
{
  "feed_id": "帖子ID",
  "xsec_token": "访问令牌"
}
```

### 步骤 5：撰写针对性评论

根据帖子内容撰写自然、有价值的评论，评论应：

- **与帖子内容相关**：根据帖子主题调整评论角度
- **突出产品价值**：强调完全免费、使用便利等特点
- **避免硬广**：采用分享经验、提醒建议的方式
- **控制长度**：简洁明了，30-80 字为宜
- **包含行动号召**：引导用户搜索或使用产品

**评论模板要素：**
```
[相关话题引入] + [产品价值点] + [使用建议/引导]
```

**示例评论（教资报名场景）：**
- 针对报名流程帖："姐妹们注意啦！报名教资还需要白底证件照哦～微信搜「过考照改图助手」小程序，完全免费，AI自动抠图换白底，1分钟搞定！"
- 针对备考攻略帖："备考计划很详细！对了提醒姐妹们，报名时需要的白底证件照别忘了准备！微信搜xxx，完全免费..."

### 步骤 6：发表评论

使用 `post_comment` 工具发表评论：

```json
{
  "feed_id": "帖子ID",
  "xsec_token": "访问令牌",
  "content": "评论内容"
}
```

### 步骤 7：更新记录文件

将新评论的帖子信息添加到已评论记录文件中：

```json
{
  "commented_posts": [
    {"feed_id": "xxx", "commented_at": "2026-01-09", "title": "...", "author": "..."}
  ],
  "failed_posts": [
    {"feed_id": "xxx", "reason": "笔记不可访问", "title": "...", "author": "..."}
  ],
  "summary": {
    "total_commented": N,
    "total_failed": M,
    "last_updated": "日期"
  }
}
```

## 最佳实践

### 评论策略

1. **个性化**：每条评论根据帖子内容调整，避免千篇一律
2. **价值导向**：突出"完全免费"、"AI自动"、"快速便捷"等卖点
3. **场景结合**：结合帖子场景（如报名、备考、认定等）给出建议
4. **时机把握**：在用户最需要的环节提供解决方案

### 常见问题处理

| 问题 | 解决方案 |
|------|----------|
| 笔记不可访问 | 记录到 failed_posts，跳过该帖子 |
| 评论频繁触发限制 | 控制评论频率，分批进行 |
| 评论内容不够自然 | 结合帖子具体内容撰写，避免模板化 |
| 重复评论 | 严格校验已评论记录，避免重复 |

### 注意事项

1. **遵守平台规则**：避免过度营销，确保评论有价值
2. **尊重原创作者**：不要在竞争对手或敏感内容下推广
3. **保护用户体验**：评论应提供有用的附加信息，而非单纯广告
4. **记录管理**：及时更新已评论记录，避免资源浪费

## 常见推广场景示例

### 场景 1：教资考试相关产品

**搜索关键词：**
- "2026年教师资格考试"
- "教资报名"
- "教资备考"

**评论角度：**
- 报名需要证件照 → 推荐证件照处理工具
- 备考需要资料 → 推荐资料获取渠道
- 认定需要材料 → 推荐材料准备工具

### 场景 2：美妆护肤产品

**搜索关键词：**
- "护肤教程"
- "化妆步骤"
- "护肤品推荐"

**评论角度：**
- 分享同款产品使用体验
- 推荐配套工具或替代品
- 解答产品相关问题

### 场景 3：工具类应用推广

**搜索关键词：**
- 工具名称 + "教程"
- 工具名称 + "替代"
- 问题解决方案 + "app"

**评论角度：**
- 遇到类似问题 → 推荐解决方案
- 分享使用心得 → 引导用户尝试
- 提供额外技巧 → 展示产品价值
