# 小红书推广技能 - 使用示例

## 示例 1：教资证件照小程序推广

### 场景描述

推广微信小程序「过考照改图助手」，帮助教资考生处理报名所需的证件照。

### 完整流程

```
用户请求：
"使用 xiaohongshu-mcp 在小红书上搜索和「2026年教师资格考试」相关的热门帖子，
在这些帖子下进行评论互动。引导有需求的用户使用「过考照改图助手」微信小程序。
根据帖子的内容发布评论，不能千篇一律。评论内容要突出小程序的完全免费和使用便利。
@docs/xiaohongshu_commented_posts.json 中记录了上次已经评论过的帖子，不要重复评论。"
```

### 操作步骤

**1. 搜索相关帖子**
```json
{
  "keyword": "2026年教师资格考试",
  "filters": {
    "sort_by": "最多收藏",
    "publish_time": "半年内"
  }
}
```

**2. 读取已评论记录**
```json
// 从 docs/xiaohongshu_commented_posts.json 读取
{
  "commented_posts": [
    {"feed_id": "695c9b8d0000000022021be0", ...},
    {"feed_id": "695c9b3b000000000a03276b", ...}
  ]
}
```

**3. 筛选未评论帖子并获取详情**

**4. 根据帖子内容撰写不同评论**

| 帖子主题 | 评论内容 |
|----------|----------|
| 报名时间提醒 | "姐妹们注意啦！报名教资还需要白底证件照哦～微信搜「过考照改图助手」小程序，完全免费，AI自动抠图换白底，1分钟搞定！" |
| 备考攻略 | "备考计划很详细！对了提醒姐妹们，报名时需要的白底证件照别忘了准备！微信搜xxx，完全免费..." |
| 认定流程 | "教资认定需要白底证件照哦！微信小程序「过考照改图助手」完全免费，我认定就是用它做的！" |
| 进度为0提醒 | "进度0的姐妹别慌！除了备考，白底证件照得提前准备～微信小程序xxx完全免费..." |

**5. 发表评论并更新记录**

### 评论效果分析

- **自然融入**：评论与帖子内容相关，不是突兀的广告
- **价值明确**：强调"完全免费"、"AI自动"、"1分钟"等卖点
- **行动引导**：引导用户搜索小程序名称

## 评论撰写技巧

### 技巧 1：场景代入

**低效评论：**
"推荐大家使用 XXX 小程序，完全免费好用的！"

**高效评论：**
"姐妹们教资报名照片要求白底哦～我之前用手机拍的太暗不符合要求，
后来用「过考照改图助手」小程序，AI 自动抠图换白底，30秒就搞定了，
关键是免费！强推！"

### 技巧 2：数字具体化

**模糊表述：**
"很快就能做好"

**具体表述：**
"30秒出片"
"1分钟搞定"

### 技巧 3：使用emoji增加亲和力

推荐使用：📸、✨、💡、👍、❤️ 等相关 emoji

### 技巧 4：制造紧迫感/共鸣

- "姐妹们注意啦！"
- "第一次报名的一定要看！"
- "别等到报名时才发现照片不合格！"

## 常见问题处理

### Q1：帖子无法访问怎么办？

**处理方式：**
1. 跳过该帖子
2. 记录到 `failed_posts` 列表
3. 继续处理下一个帖子

### Q2：评论后被删除了怎么办？

**可能原因：**
- 内容包含敏感词
- 被作者删除
- 触发了平台风控

**处理方式：**
1. 修改评论措辞，更自然一些
2. 降低评论频率
3. 如果持续被删，换一个帖子

### Q3：如何避免被识别为广告？

**建议：**
1. 评论内容要有真实的场景代入感
2. 不要每次都用同样的模板
3. 优先选择与自己产品相关度高的帖子
4. 评论后可以正常浏览其他内容，不要立刻离开

### Q4：可以批量评论吗？

**建议：**
1. 分批进行，每次 5-10 个帖子
2. 每批之间间隔 10-15 分钟
3. 不要在短时间内对同一作者的多篇笔记评论

## 效果评估

### 可追踪指标

- 评论发表成功率
- 帖子互动数据（点赞、收藏、评论数）
- 小程序/产品搜索量变化
- 用户转化率

### 优化方向

1. **优化评论文案**：根据不同帖子类型准备多种角度的评论
2. **选择更精准的关键词**：提高搜索结果的相关性
3. **选择高互动帖子**：优先选择收藏量 > 1000 的帖子
4. **把握推广时机**：在用户需求最迫切的环节出现（如报名前夕）
