# 小红书爆款笔记创作

## 功能概述

此技能帮助用户完成小红书内容创作的完整流程：
1. 搜索分析热门笔记
2. 总结爆款规律
3. 创作符合规律的内容
4. 发布笔记到小红书

## 使用方式

当用户需要创作小红书笔记时，使用此技能：

```
/xiaohongshu-content-creator
```

或者描述创作需求，如：
- "帮我发个小红书"
- "创作一篇关于xxx的笔记"
- "用这个素材发小红书"

**首次使用前**，请先运行准备脚本完成登录和启动：

```bash
cd ~/.claude/skills/xiaohongshu-content-creator && ./scripts/prepare.sh
```

此脚本会自动：
1. 检查是否已登录，未登录则启动登录流程
2. 启动 xiaohongshu-mcp 服务

## 操作流程

### 第一步：准备素材
- 用户提供图片/视频素材
- 说明创作主题或方向

### 第二步：搜索并深度分析热门内容（关键步骤！）

**搜索阶段**：
使用 `search_feeds` 搜索相关热门笔记：
- 关键词搜索：主题相关词
- 筛选条件：按点赞数排序，发布时间半年内
- 获取前5-10篇高赞笔记的 id 和 xsec_token

**深度分析阶段（必须执行！）**：
对排名前3-5的高赞笔记，使用 `get_feed_detail` 获取完整内容，包括：
- 笔记正文内容
- 图片/视频内容
- 评论列表（了解互动点）

**分析爆款规律时需记录**：
1. **标题公式**：分析标题用了什么技巧（数字/悬念/对比/情绪词/疑问）
2. **开头钩子**：首句话如何吸引继续阅读
3. **内容结构**：如何展开（分点/时间线/对比/故事）
4. ** emoji 使用**：哪些emoji出现频率高，放在什么位置
5. **段落节奏**：长句还是短句，分段频率
6. **结尾互动**：如何引导评论/点赞/收藏
7. **标签组合**：标签数量、类型、排列顺序
8. **情感钩子**：引发什么情绪（焦虑/期待/共鸣/优越感）

**输出分析结果**：
在创作前，先向用户展示分析总结：
```
📊 爆款规律分析：
- 标题特点：xxx
- 开头模式：xxx
- 内容结构：xxx
- 结尾互动：xxx
- 高频emoji：xxx
```

### 第三步：向用户展示分析结果

在创作前，先向用户展示分析总结，格式如下：
```
📊 爆款规律分析（基于X篇高赞笔记）：

🎯 标题模式：
- 主要技巧：xxx
- 推荐公式：xxx

📝 开头钩子：
- 常见开头：xxx
- 借鉴写法：xxx

💡 内容结构：
- 主流结构：xxx
- 段落特点：xxx

🎨 视觉元素：
- 高频emoji：xxx
- 格式特点：xxx

💬 结尾互动：
- 常见方式：xxx
- 推荐引导：xxx

🏷️ 标签策略：
- 标签数量：xxx
- 标签组合：xxx

基于以上分析，建议创作方向：xxx
```

**等待用户确认后继续**。

### 第四步：创作笔记内容
根据分析结果创作，**必须严格遵循发现的规律**：

**标题**：直接套用发现的标题公式，包含关键技巧元素

**正文**：
- 开头：使用分析中发现的钩子模式
- 结构：模仿高分内容的展开方式
- 节奏：参考段落长度和emoji使用习惯
- 结尾：使用分析中有效的互动引导方式

**话题标签**：参考热门笔记的标签组合方式和数量

### 第五步：发布笔记
使用 `publish_content` 或 `publish_with_video` 发布：
- 图文内容：publish_content
- 视频内容：publish_with_video

## 必备工具

此技能需要以下 xiaohongshu-mcp 工具：
- `check_login_status` - 检查登录状态
- `search_feeds` - 搜索热门内容（第一步）
- `get_feed_detail` - 获取笔记详情（关键步骤！必须获取前3-5篇高赞笔记的详细内容）
- `publish_content` - 发布图文
- `publish_with_video` - 发布视频

## 内容创作模板

### 摄影类笔记模板

**标题**：
- 悬念式："清晨六点的xxx，月亮还舍不得走"
- 数字式："3个技巧教你拍出xxx"
- 情感式："在xxx，我找到了内心的平静"

**正文结构**：
1. 开头（场景引入，制造悬念）："早起去拍日出，没想到..."
2. 中间（主体内容，情感描写）："站在那里，看着xxx，那一刻..."
3. 实用Tips（增加价值）：📍位置、⏰时间、📷设备
4. 结尾（互动引导）："你们有没有拍到过..."

**话题标签**：
- 主题标签：#xxx摄影 #旅行xxx
- 场景标签：#清晨 #日落 #夜景
- 平台标签：#小红书摄影 #城市探索

## 示例对话

**用户**："帮我用这张照片发个小红书"

**Claude**：
1. 读取用户图片
2. 搜索相关热门笔记（search_feeds）
3. 对前5篇高赞笔记获取详情（get_feed_detail）
4. 分析内容：标题公式、开头钩子、内容结构、emoji使用、结尾互动
5. 向用户展示分析结果，等待确认
6. 基于分析结果创作笔记内容
7. 发布笔记

**错误示范**（严禁）：
- 只搜索标题，不获取详细内容
- 凭感觉创作，不参考爆款规律
- 不展示分析过程，直接发布

## 注意事项

- **自动准备**：首次使用技能时，会自动检查登录状态并启动 MCP 服务
- 发布前检查登录状态
- 图片路径使用绝对路径
- 标题限制20字以内
- 标签选择与内容高度相关
- 保持内容真实性和个人特色

**⚠️ 核心原则**：
- **绝对禁止**不获取详细内容就直接创作
- 必须用 `get_feed_detail` 分析至少3篇高赞笔记
- 分析结果必须展示给用户看
- 创作内容必须严格遵循发现的规律
- 不要凭感觉创作，要用数据驱动
