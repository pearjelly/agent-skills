# 小红书爆款笔记创作

## 功能概述

此技能帮助用户完成小红书内容创作的完整流程：
1. 搜索分析热门笔记
2. 总结爆款规律
3. 创作符合规律的内容
4. 发布笔记到小红书

## 使用方式

当用户需要创作小红书笔记时，使用此技能：

```
/xiaohongshu-content-creator
```

或者描述创作需求，如：
- "帮我发个小红书"
- "创作一篇关于xxx的笔记"
- "用这个素材发小红书"

**首次使用前**，请先运行准备脚本完成登录和启动：

```bash
cd ~/.claude/skills/xiaohongshu-content-creator && ./scripts/prepare.sh
```

此脚本会自动：
1. 检查是否已登录，未登录则启动xiaohongshu-login登录流程
2. 启动 xiaohongshu-mcp 服务

## 操作流程

### 第一步：准备素材
- 用户提供图片/视频素材
- 说明创作主题或方向

### 第二步：搜索分析热门内容
使用 `search_feeds` 搜索相关热门笔记：
- 关键词搜索：主题相关词
- 筛选条件：按点赞数排序，发布时间半年内
- 获取多篇高赞笔记详情

### 第三步：分析爆款规律
从热门笔记中总结：
- 标题特点：悬念式、数字式、情感式等
- 内容结构：开头悬念、中间叙事、结尾互动
- 话题标签：热门标签组合
- 情感共鸣点

### 第四步：创作笔记内容
根据分析结果创作：
- **标题**：符合爆款规律，吸引点击
- **正文**：故事性开头 + 场景描写 + 实用价值 + 互动引导
- **话题标签**：选择热度高的相关标签

### 第五步：发布笔记
使用 `publish_content` 或 `publish_with_video` 发布：
- 图文内容：publish_content
- 视频内容：publish_with_video

## 必备工具

此技能需要以下 xiaohongshu-mcp 工具：
- `list_feeds` - 获取首页动态
- `search_feeds` - 搜索热门内容
- `get_feed_detail` - 获取笔记详情
- `publish_content` - 发布图文
- `publish_with_video` - 发布视频
- `check_login_status` - 检查登录状态

## 内容创作模板

### 摄影类笔记模板

**标题**：
- 悬念式："清晨六点的xxx，月亮还舍不得走"
- 数字式："3个技巧教你拍出xxx"
- 情感式："在xxx，我找到了内心的平静"

**正文结构**：
1. 开头（场景引入，制造悬念）："早起去拍日出，没想到..."
2. 中间（主体内容，情感描写）："站在那里，看着xxx，那一刻..."
3. 实用Tips（增加价值）：📍位置、⏰时间、📷设备
4. 结尾（互动引导）："你们有没有拍到过..."

**话题标签**：
- 主题标签：#xxx摄影 #旅行xxx
- 场景标签：#清晨 #日落 #夜景
- 平台标签：#小红书摄影 #城市探索

## 注意事项

- **自动准备**：首次使用技能时，会自动检查登录状态并启动 MCP 服务
- 发布前检查登录状态
- 图片路径使用绝对路径
- 标题限制20字以内
- 标签选择与内容高度相关
- 保持内容真实性和个人特色
